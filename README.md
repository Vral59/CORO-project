# Project Cutting stock problem

Project made by JINZhao2000 and I

## Environnement
julia version 1.8.5
libraries:
- JuMP
- SparseArrays
- GLPK
- LinearAlgebra
- BenchmarkTools

System: Ubuntu 22.04.2 LTS jammy
RAM: 16G
CPU: Intel i7-1165G7 8 cores

## Use
- Install libraries
```bash
julia ./requirement.jl
```
- Run program
```bash
julia ./project.jl
```
- Run benchmark
```bash
julia ./project.jl  BM
```
